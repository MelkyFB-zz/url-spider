# url-spider

```
URL Spider - A Web Spider
Usage:
    
java -jar url_spider.jar <options> url

Options:
--maxdepth=N (N default=3) catch N depth for url
-o=PATH --output=PATH (Default to console) write to an output file 
-h --help show this usage
```
